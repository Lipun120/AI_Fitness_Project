FitGen AI is an AI-powered fitness and nutrition planning platform that delivers personalized workout routines and diet plans based on user goals, preferences, and health data. 
The system is built using a scalable microservices architecture with secure authentication and event-driven communication.

Features :-

-> AI-generated personalized workout & diet plans

-> Secure authentication with OAuth2 & JWT (Keycloak)

-> Microservices-based architecture for scalability

-> Event-driven communication using Apache Kafka

-> Centralized routing with Spring Cloud Gateway

-> Role-based access control (Admin / User)

-> Fault-tolerant and highly available services

Tech Stack :-

-> Backend

-> Java 8+

-> Spring Boot

-> Spring MVC

-> Spring Cloud (Gateway, Config)

-> Hibernate & JPA

-> Microservices Architecture

-> Apache Kafka

-> Security

-> Keycloak

-> OAuth2

-> JWT Authentication

-> AI Integration

-> Gemini API (AI-based fitness & nutrition recommendations)

-> Database

-> MySQL / PostgreSQL

Tools & Platforms:-

-> Maven

-> Git & GitHub
